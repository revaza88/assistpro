## ­ЪЊІ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ

AssistPro рЃљрЃарЃўрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЋрЃћрЃЉ-рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃўрЃљ рЃЎрЃЮрЃарЃърЃЮрЃарЃљрЃбрЃўрЃБрЃџрЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЏрЃгрЃЮрЃЊрЃћрЃЉрЃћрЃџрЃў рЃЎрЃЮрЃЏрЃърЃљрЃюрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃАрЃљрЃўрЃбрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃАрЃарЃБрЃџ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃЎрЃЮрЃЏрЃърЃљрЃюрЃўрЃўрЃА рЃЏрЃўрЃћрЃа рЃерЃћрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃБрЃџ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃќрЃћ, рЃњрЃБрЃюрЃЊрЃўрЃА рЃгрЃћрЃЋрЃарЃћрЃЉрЃќрЃћ рЃЊрЃљ рЃАрЃљрЃЎрЃЮрЃюрЃбрЃљрЃЦрЃбрЃЮ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃќрЃћ.

## ­ЪЊџ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

### ­ЪЈб рЃЉрЃўрЃќрЃюрЃћрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў
- **рЃАрЃљрЃЉрЃБрЃдрЃљрЃџрЃбрЃарЃЮ рЃЏрЃЮрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ** - рЃАрЃарЃБрЃџрЃў рЃцрЃўрЃюрЃљрЃюрЃАрЃБрЃарЃў рЃЏрЃљрЃарЃЌрЃЋрЃљ
- **рЃАрЃљрЃњрЃљрЃЊрЃљрЃАрЃљрЃ«рЃљрЃЊрЃЮ рЃЎрЃЮрЃюрЃАрЃБрЃџрЃбрЃљрЃфрЃўрЃљ** - рЃАрЃљрЃњрЃљрЃЊрЃљрЃАрЃљрЃ«рЃљрЃЊрЃЮ рЃбрЃЋрЃўрЃарЃЌрЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљ
- **рЃўрЃБрЃарЃўрЃЊрЃўрЃБрЃџрЃў рЃЏрЃЮрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ** - рЃЎрЃљрЃюрЃЮрЃюрЃўрЃћрЃарЃў рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ
- **рЃЉрЃўрЃќрЃюрЃћрЃА-рЃљрЃАрЃўрЃАрЃбрЃћрЃюрЃбрЃў** - рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃБрЃџрЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ
- **рЃљрЃБрЃЊрЃўрЃбрЃў** - рЃцрЃўрЃюрЃљрЃюрЃАрЃБрЃарЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў
- **HR рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў** - рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃБрЃарЃў рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ

### ­ЪћД рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў
- **рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў** - рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
- **SEO рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљ** - рЃФрЃўрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
- **рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃў** - рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃљ
- **рЃЉрЃЏрЃБрЃџрЃў рЃцрЃЮрЃарЃЏрЃћрЃЉрЃў** - рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА
- **рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃљ** - рЃерЃўрЃюрЃљрЃљрЃарЃАрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
